(async, inner) loginUser(email, password, role) → {Promise.<{id: string, name: string, email: string, role: string}>}

Authenticate a user against the Supabase `users` table. The query checks: - email - role - password_hash If a matching user is found, a minimal user object is returned and the session is stored in localStorage.
